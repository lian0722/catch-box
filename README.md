# 玄学盲盒抽取大师

一个基于AI的玄学盲盒预测网页应用。


## 📱 功能特性

- 🎯 玄学盲盒预测
- 🔮 AI智能分析
- 📤 一键分享
- 📱 响应式设计
- 🔒 后台API配置

## 🎨 自定义

你可以修改以下部分来自定义应用：

1. **UI样式** - 修改 `index.html` 中的CSS
2. **AI提示词** - 修改JavaScript中的prompt构建部分
3. **API配置** - 修改 `config.js` 文件

## 📞 支持

如有问题，请检查：
1. API Key是否正确配置
2. 网络连接是否正常
3. API配额是否充足
